Accellerating Augmented Reality Video Processing With FPGAs
Authors: Anthony Dresser, Lukas Hunker, Andrew Weiler

In order to build this project, open Vivado 2015.2 and run the tcl script in
the root folder called build.tcl. This will construct the entire project under
the folder armqp.

In order to access the sdk, open the project located in src/sdk in Xilinx SDK
2015.2.
